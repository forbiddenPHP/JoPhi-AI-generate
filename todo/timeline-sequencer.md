# Timeline/Sequencer-Engine

Mehrere generierte Snippets (Audio, später Video) über eine Timeline-Definition zusammenführen. Jedes Segment separat generieren, dann via ffmpeg zusammenführen.

## Offene Fragen

- Format der Timeline-Definition (JSON/YAML?)
